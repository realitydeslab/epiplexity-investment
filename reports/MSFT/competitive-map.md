## Who Loses if MSFT Wins?

- **GOOGL** — Every enterprise Copilot seat is a lost Workspace opportunity
- **IBM** — Legacy enterprise IT displacement accelerates
- **ORCL** — Azure wins in cloud migration market
- **On-prem software vendors** — Cloud migration tailwind benefits MSFT

## Pair Trade Opportunities

1. **Long MSFT / Short IBM** — AI leader vs. legacy IT. MSFT growing 17%, IBM stagnant.
2. **Long MSFT / Short ORCL** — Both enterprise, but MSFT has broader AI moat.
3. **Long MSFT + NVDA** (correlated pair) — The core AI infrastructure stack.

## Competitive Moat Assessment

- **Distribution moat:** 400M+ M365 users = built-in Copilot distribution
- **Data moat:** Enterprise data in Azure/M365 feeds AI fine-tuning
- **Ecosystem moat:** GitHub (100M+ developers) + LinkedIn (1B+) + Azure = full-stack AI
- **Partnership moat:** OpenAI exclusive cloud provider
- **Financial moat:** $3T market cap, $305B revenue, $119B net income = can outspend anyone

**Verdict:** MSFT has the widest competitive moat in AI. The only real threat is if open-source models (Llama, etc.) commoditize the model layer — but MSFT wins on infrastructure regardless.
